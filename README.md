# Telegram daily news digest

Этот проект собирает новости из RSS-лент по темам:
- бизнес
- ИТ и enterprise-рынок
- маркетинг
- российский и глобальный ландшафт

Он формирует короткий дайджест в формате:
- одна новость = одна строка
- есть тезис и ссылка на источник
- можно отправлять в Telegram каждый день

## Как запустить

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Создайте Telegram-бота через BotFather.
   Получите токен и ID чата.

3. Укажите переменные окружения:
   ```bash
   export TELEGRAM_BOT_TOKEN="..."
   export TELEGRAM_CHAT_ID="..."
   ```

4. Проверьте локально без отправки:
   ```bash
   python news_bot.py --dry-run --limit 8
   ```

5. Отправьте в Telegram:
   ```bash
   python news_bot.py --limit 8
   ```

## Бесплатный автозапуск

Проект уже настроен под GitHub Actions. Достаточно:
- добавить Secrets в репозиторий:
  - `TELEGRAM_BOT_TOKEN`
  - `TELEGRAM_CHAT_ID`
- включить Actions
- запуск будет происходить ежедневно в 08:00 UTC
